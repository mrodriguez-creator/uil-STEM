# UIL STEM Academics

Practice tools for Texas UIL academic competitions. Built as a static web app — no server required, works in any browser.

**Live site:** [mrodriguez-creator.github.io/uil-STEM](https://mrodriguez-creator.github.io/uil-STEM/)

## Events

| Event | Problems | Format | Status |
|-------|----------|--------|--------|
| **Number Sense** | 80 problems | Mental math, 10 min, no calculator | Active |
| **Science** | Biology, Chemistry, Physics | 60 questions, 30 min | Active |
| **Calculator Applications** | Numerical, Stated, Geometry | 70 problems, 30 min | Active |
| **Mathematics** | 310 problems across 7 topics | 60 questions, 40 min, 5-choice (A-E) | Active |
| **Computer Science** | — | — | Coming Soon |

## Mathematics Topics

Algebra, Geometry, Trigonometry, Pre-Calculus, Calculus, Statistics, and Word Problems.

Each problem includes:
- 5 answer choices (A-E)
- Hint (collapsible)
- Detailed explanation (shown after answering)
- Difficulty rating (1-5 stars)

### Practice Modes

- **Skills** — Practice by topic
- **Drill** — Random questions across all topics
- **Practice Test** — Timed test simulating UIL competition (60 questions, 40 minutes)

### Difficulty Filter

Filter problems by difficulty (1-5 stars) in Skills and Drill modes. Practice Test uses all questions to simulate real competition conditions.

## Running Locally

Open `index.html` in a browser. No build step or dependencies needed.

## Tech Stack

Vanilla HTML, CSS, and JavaScript. No frameworks. Data stored in `localStorage` for stats persistence.
